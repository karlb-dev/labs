# Lab 25 Introspection Query Corpus

`introspection_queries.csv` is a frozen, deterministic prompt set for Lab 25.
It is generated by `make_introspection_queries.py` and contains 24 benign rows:
eight target concepts, three rows per concept, with two train rows and one
held-out eval row per concept.

The corpus is designed to avoid the main failure mode in earlier Lab 25 runs:
report prompts no longer name the target and wrong concepts. Positive and
negative contrast prompts still name the concepts because they are used only to
build local directions. The open report prompt asks for a short state label or
`none`; the behavior prompt is neutral so visible-output expression can be
measured separately.

Families:

- `emotion`: `joy`, `calm`
- `register`: `technical_register`, `plain_register`
- `voice`: `warm_supportive_voice`, `playful_voice`
- `topic`: `weather_topic`, `gardening_topic`

Known limitations:

- Automatic scoring is lexical and must be hand-checked before any strong
  self-report claim.
- The concepts are benign and intentionally coarse. The lab tests whether this
  instrument can find a narrow state-coupled report channel, not whether models
  have human-like introspection.
- The source-attribution task remains a high bar: `activation_injection` is a
  hidden cause, so failure after this repair is a meaningful negative rather
  than a prompt-leak artifact.
