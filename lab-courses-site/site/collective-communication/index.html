01 Course premise

Communication is
ownership in motion.

A collective is more than an API call. It is a schedule: who owns each chunk, where it moves, when it becomes safe to read, and how many bytes cross every link.

Make the topology, synchronization, and byte model visible—then measure the implementation against all three.

02 Curriculum arc

One hop to
the whole fabric.

Each phase preserves the invariants from the one before it. Local movement becomes a collective; the collective becomes topology-aware; the schedule becomes a measured claim.
